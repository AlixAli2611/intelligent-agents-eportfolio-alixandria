Individual Implementation: ResearchMate

My individual implementation developed ResearchMate into a working LLM-powered academic research planning agent.

Implemented Workflow

  • User enters topic, purpose, audience level, and requested number of papers.
  • Planning Agent creates an ordered plan.
  • Retrieval Agent queries arXiv, Semantic Scholar, and OpenAlex.
  • Processing Agent creates summaries, key terms, and paper type labels.
  • Ranking Agent scores papers and assigns recommendation labels.
  • Evidence assessment produces a cautious evidence note.
  • Storage saves Markdown, JSON, and SQLite outputs.
