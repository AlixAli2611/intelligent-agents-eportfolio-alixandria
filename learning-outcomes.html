Learning Outcomes

This section maps my module evidence to the Intelligent Agents learning outcomes.

LO1: Identify and critically analyse agent-based systems

I developed this outcome through the ResearchMate group proposal, where we compared single-agent and multi-agent approaches and selected a modular multi-agent architecture.

LO2: Apply and critically evaluate intelligent agent techniques

I applied agent techniques through the individual ResearchMate prototype, which receives a research goal, plans actions, retrieves academic data, processes information, ranks sources, and saves outputs.

LO3: Deploy appropriate software tools ethically and professionally

The project used Python, Groq/Llama, Pydantic, SQLite, pytest, and academic APIs while considering hallucination risk, transparency, source limits, and human review.

LO4: Work effectively as part of a virtual development team

The group project developed my teamwork, communication, role negotiation, and resilience in a virtual environment.
