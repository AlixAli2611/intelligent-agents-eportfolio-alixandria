Testing and Remediation

The final implementation was tested using pytest and produced 24 passing tests.

Testing Covered

  • Planning agent behaviour
  • Processing agent output
  • Ranking and recommendation logic
  • Evidence assessment
  • Markdown and JSON output generation
  • SQLite persistence
  • LLM JSON validation and fallback logic

Remediation Activities

  • Ranking tests were refined to check meaningful behaviour rather than exact LLM wording.
  • Groq JSON parsing was improved to handle markdown fences or extra text.
  • Retrieval behaviour was improved to handle Semantic Scholar rate limits.
  • An accidental nested clone causing duplicate test discovery was removed.
  • SQLite persistence was added and tested.
